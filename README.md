# ᚛ᚑᚌᚆᚐᚋ᚜ 
**Ogham** ( ᚛ᚑᚌᚆᚐᚋ᚜ ) is a cross-platform widget manager for the terminal. 

## Features 
- Responsive
- Terminal colors
- UTF-8 support
- Only standart libs (no dependencies)

# How to install
**Ogham** is a header-only library. So just copy the `ogham.hpp` file in your project, and enjoy ^^
